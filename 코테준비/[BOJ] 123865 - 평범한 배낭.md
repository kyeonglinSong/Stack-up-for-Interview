# [BOJ] 123865번 - 평범한 배낭

### 1. 구조

- 사용한 자료구조 : 그냥배열
- 사용한 알고리즘 : 메모이제이션 + dfs (dp)

그냥 dfs를 사용하면 시간초과가 나버린다.

  

### 2. 코드

```c++
#include <iostream>
#include <vector>
#include <cstring>
using namespace std;

int num, weight;
int map[100][100001];
vector<pair<int,int>> items;

int dfs()
{
    for(int i=1 ; i<=weight ; i++)
    {
        if(items[0].first <= i)
            map[0][i] = items[0].second;   
    }

    for(int i=1 ; i<num ; i++)
    {
        for(int j=1 ; j<=weight ; j++)
        {
            if(items[i].first <= j)
                map[i][j] = max(map[i-1][j], map[i-1][j-items[i].first]+items[i].second);
            else
                map[i][j] = map[i-1][j];
        }
    }
    return map[num-1][weight];
}

int main()
{
    memset(map, 0, sizeof(map));
    cin >> num >> weight;
    int wei, val;
    for(int i=0 ; i<num ; i++)
    {
        cin >> wei >> val;
        items.push_back(make_pair(wei, val));
    }
    cout << dfs();
    
}

```
