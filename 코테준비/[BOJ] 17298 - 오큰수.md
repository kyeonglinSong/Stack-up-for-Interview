# [BOJ] 17298번 - 오큰수

 

### 1. 구조

- 사용한 자료구조 : 스택
- 사용한 알고리즘 : X



스택을 이용하는 문제.

결과 배열을 모두 -1로 초기화하고 처음부터 끝까지 for문을 통해 오큰수를 찾아간다.





### 2. 알고리즘

(1) 스택을 하나 선언

(2) 스택이 비엇거나 top()이 현재 값보다 크다면, 현재 인덱스를 스택에 집어넣는다.

(3) top()이 현재 값보다 작다면, 현재 값을 결과 배열에 집어넣는다.

(4) pop()을 하고 2번이나 3번으로 돌아간다.

(5) 결과 배열을 출력한다.





### 3. 코드

```c++
#include <iostream>
#include <cstring>
#include <stack>
using namespace std;

int n;
int arr[1000000];
int result[1000000];

void rbn()
{
    stack<int> st;

    for(int i=0 ; i<n ; i++)
    {
        while(!st.empty() && (arr[st.top()] < arr[i]) )
        {
            result[st.top()] = arr[i];
            st.pop();
        }
        st.push(i);
    }

    for(int i=0 ; i<n ; i++)
        cout << result[i] << " ";
}
int main()
{
    cin >> n;

    if(n == 1)
    {
        cout << -1;
    }
    else
    {
        for(int i=0 ; i<n ; i++)
            cin >> arr[i];

        memset(result, -1, sizeof(result));
        rbn();
    }
    
    return 0;
}
```
